# Projeto Spring Batch com duas fontes de dados e RabbitMQ

Este projeto demonstra a configuração e implementação de um aplicativo Spring Batch que 
utiliza duas fontes de dados diferentes, sendo uma para leitura dos dados e outra para 
gravar os metadados de execução dos jobs do Spring. Além disso, as informações lidas são 
enviadas para uma fila do RabbitMQ.
